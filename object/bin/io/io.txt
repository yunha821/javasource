I/O (input/output) : 컴퓨터 내부 혹은 외부의 장치와 프로그램간의 데이터를 주고 받기

System.out : 화면
System.in : 키보드

파일(text.txt) 에서 text를 읽어서 프로그램에서 읽어온 텍스트 사용하기
프로그램의 결과를 파일로 작성

스트림 : 데이터를 운반하는 데 사용하는 연결 통로(단방향만 가능)

1) 바이트 기반 스트림
- 바이트 단위로 데이터 전송
- 그림, 동영상, 텍스트 ... 모든 데이터 가능

-InputStream(입력 스트림)
  - read() / read(byte[] b) / read(byte[] b, int off, int len)
  -FileInputStream, PipeInputStream, AudioInputStream...
-OutputStream(출력 스트림)
  - write(int b) / write(byte[] b) /write(byte[] b, int off, int len)
  - FileOutputStream...


  - File : 가장 많이 사용되는 입출력 대상
   - File, 디렉토리 다룰 수 있음

2) 문자 기반 스트림
- 문자 데이터
- Reader
  -FileReader..
- Writer
  -FileWriter..


보조 스트림
바이트 기반 / 문자 기반 + 보조스트림
- 바이트 기반 + BufferedInputStream / BufferedOutputStream
- 문자 기반 + BufferedReader / BufferedWriter


직렬화(Serialization)
- 객체를 데이터 스트림으로 만드는 것
- transient : 멤버 변수 저장하지 않기

역직렬화(Deserialization)
- 스트림으로부터 데이터를 읽어서 객체로 만드는 것
